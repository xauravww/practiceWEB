# Piano Mini Project

- This mini project is a simple piano implementation using HTML, CSS, and JavaScript.
- It allows users to play different musical notes by pressing corresponding keys on the keyboard.

## Live Demo

- Try on PC only , not designed for smaller screens , works with keyboard only
- Check out the live demo of the Piano Mini Project at
https://xauravww-piano.netlify.app/

## Features

- Interactive piano interface.
- Play musical notes by pressing keys on the keyboard.
- Visual feedback for active keys.

## Technologies Used

- HTML: Provides the basic structure of the web page.
- CSS: Styles the piano for an engaging user interface.
- JavaScript: Drives the piano's functionality, handling key presses and audio playback.

## Use Locally

To use this piano project locally, clone the repository using the following Git command:

`git svn clone https://github.com/xauravww/practiceWEB/trunk/mini-projects/adv-midi-piano-miniproject`

Feel free to make enhancements, customize the piano keys, or report any issues you encounter. Enjoy creating music with the virtual piano!
