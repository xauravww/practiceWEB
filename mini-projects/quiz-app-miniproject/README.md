# Quiz App Mini Project

This Quiz App is a simple web application designed for testing your knowledge on various topics. It's built using HTML, CSS, and JavaScript.

## Live Demo
Check out the live demo of the Quiz App at 
https://xauravww-quiz-app.netlify.app/

## Features

- Multiple-choice quiz questions.
- Immediate feedback on correct and incorrect answers.
- Elegant and user-friendly responsive design.

## Technologies Used

- **HTML:** Provides the structure of the quiz.
- **CSS:** Styles the quiz for a visually appealing look.
- **JavaScript:** Handles quiz functionality, real-time feedback, and submission.
