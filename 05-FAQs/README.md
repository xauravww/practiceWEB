# FAQs App

This React application is a simple Frequently Asked Questions (FAQs) app that allows users to explore and find answers to common queries. The app utilizes React components to create a user-friendly interface for interacting with the FAQs.


![Project Gif](https://res.cloudinary.com/drvntsbpo/image/upload/v1706003678/FAQs_tnmrxc.webp)

This project is live on [https://xauravww-faqs.netlify.app/](https://xauravww-faqs.netlify.app/).

## Features

- Display frequently asked questions and their corresponding answers.
- Allow users to expand and collapse individual FAQ items.
- Use React components for a modular and organized structure.
- Simple and intuitive user interface.
- Fully Responsive Design

## Technologies Used

- ReactJS
- CSS

## Usage

1. Open the app in your browser.
2. Explore the list of frequently asked questions.
3. Click on an FAQ item to expand and view its answer.
4. Click again to collapse the answer.

## Installation

#### Step 1: Clone the subdirectory of this repository
```git svn clone https://github.com/xauravww/practiceWEB/trunk/react-projects/05-FAQs/```

#### Step 2: Install the dependencies
```npm install```

#### Step 3: Start the server
```npm run dev```

#### Step 4: Open the webapp
 Open `http://localhost:4173/` in your browser to view the app.


## Folder Structure

- `src/`: Contains the main source code for the React app.
- `Components/:` Holds React components used in the app.
    - `Box/:` Component for displaying the FAQ box.
    - `Item/:` Component for rendering individual FAQ items.
- `utils/:` Contains utility files such as data.js for storing FAQ data.
- `App.js:` Main component representing the entire application.
