# Menu App

This React application is a simple menu app that displays a variety of vegetarian dishes categorized into breakfast, lunch, snacks, and dinner. Users can filter the menu items based on these categories, providing an easy-to-navigate experience , with dynamic buttons according to data provided.


![Project Gif](https://res.cloudinary.com/drvntsbpo/image/upload/v1706041975/Menu-optimized_oy6wfu.webp)

This project is live on [https://xauravww-menu.netlify.app/](https://xauravww-menu.netlify.app/).

## Features

- Categorizes vegetarian dishes into breakfast, lunch, snacks, and dinner.
- Dynamic creation of buttons according to data provided.
- Allows users to filter menu items by clicking on category buttons.
- Utilizes React components for a modular and organized structure.
- Displays dish details such as title, price, image, and description.
- Responsive design for optimal viewing on various devices.

## Technologies Used

- ReactJS
- CSS

## Usage

1. Open the app in your browser.
2. Explore the menu items categorized into different types.
3. Click on the category buttons to filter items accordingly.
4. View details of each menu item, including title, price, image, and description.

## Installation

#### Step 1: Clone the subdirectory of this repository
```git svn clone https://github.com/xauravww/practiceWEB/trunk/react-projects/07-Menu/```

#### Step 2: Install the dependencies
```npm install```

#### Step 3: Start the server
```npm run dev```

#### Step 4: Open the webapp
 Open `http://localhost:4173/` in your browser to view the app.


## Folder Structure

- `src/`: Contains the main source code for the React app.
- `Components/:` Holds React components used in the app.
    - `Item/:`  Component for rendering individual menu items.
- `utils/:` Contains utility files such as data.js for storing menu data.
- `App.js:` Main component representing the entire application.
