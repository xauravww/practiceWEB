# Educational Experience App

This React application allows users to explore school details, duration, and marks. The app provides a user-friendly interface to navigate through educational experiences.
This can be useful in portfolio websites to showcase experiences.

## Features

- Displays educational experiences with details such as class, school, duration, and average marks.
- Allows users to filter educational experiences based on different classes dynamically without explicitly setting button for each experience
- Utilizes React components for a modular and organized structure.
- Responsive design for optimal viewing on various devices.

- ReactJS
- CSS
- Vite (for building and bundling)

## Installation

#### Step 1: Clone the subdirectory of this repository

`git svn clone https://github.com/xauravww/practiceWEB/trunk/react-projects/08-Educational-Experience/`

#### Step 2: Install the dependencies

`npm install`

#### Step 3: Start the server

`npm run dev`

#### Step 4: Open the webapp

Open `http://localhost:4173/` in your browser to view the app.

## Usage

1. Open the app in your browser.
2. Explore different classes by clicking on the corresponding buttons.
3. View details of educational experiences, including school name, class, duration, and marks obtained.

## Folder Structure

- `src/`: Contains the main source code for the React app.
- `Components/:` Holds React components used in the app.
  - `Item/:` Component for rendering individual educational experiences.
- `utils/:` Contains utility files such as data.js for storing educational data.
- `App.js:` Main component representing the entire application.
